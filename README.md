# NaturEspaces — Site internet

Site institutionnel de **NATURESPACES**, SARL d'insertion paysagiste basée à Chaville (Hauts-de-Seine).

« Pensons le paysage autrement »

## À propos

NaturEspaces est une entreprise d'insertion par l'activité économique (EI), labellisée Économie Sociale et Solidaire (ESS). Depuis 2013, l'entreprise conjugue exigence paysagère et engagement social, intervenant sur 106 sites pour 19 clients institutionnels et privés en Île-de-France.

## Contenu du site

- `index.html` — Page d'accueil (présentation, références, contact)
- `mentions-legales.html` — Mentions légales conformes LCEN
- `politique-confidentialite.html` — Politique de confidentialité conforme RGPD
- `assets/` — Images, styles CSS et scripts

## Stack technique

- HTML5 statique
- TailwindCSS via CDN
- Polices Google Fonts (Playfair Display, Outfit)
- Icônes Material Symbols
- JavaScript vanilla (aucune dépendance externe)

## Hébergement

Site déployé sur Gandi Simple Hosting. La présente version GitHub Pages sert de preview de validation.
